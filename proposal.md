## What will (likely) be the title of your project?

DNA Sequence Analyzer and Mutation Detector

## In just a sentence or two, summarize your project. (E.g., "A website that lets you buy and sell stocks.")

This project is a Python program that analyzes DNA sequences by counting nucleotides, converting DNA to RNA and protein, and identifying possible mutations. It demonstrates how genetic data can be processed in genomic medicine.

## In a paragraph or more, detail your project. What will your software do? What features will it have? How will it be executed?

My project will be a Python-based DNA sequence analyzer that allows users to input genetic sequences and receive meaningful biological outputs. The program will count nucleotide frequency (A, T, C, G), transcribe DNA into RNA, and translate RNA into a protein sequence using codon mapping. In addition, the program will include a mutation detection feature that compares two DNA sequences and identifies differences, such as substitutions. The software will run in a command-line interface where users can input sequences and receive clear, labeled outputs. I plan to implement this using basic Python concepts such as loops, conditionals, strings, and dictionaries, and possibly use Biopython if allowed. The goal is to simulate how genomic data is analyzed in real-world medical and research settings.
